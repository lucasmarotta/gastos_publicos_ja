Sobre Dados Abertos Já

Dados Abertos Já é uma inciativa para organizar as informações de despesas dos estados da federação. Para obtermos esses dados são consultados os dados abertos dos portais de transparência dos estados. Alguns estados não possuem informações atualizadas disponíveis.
